### 💼 Business Topic
**Iwan (إيوان)** is a premium Real Estate Extranet designed for the high-end UAE and Saudi markets. It digitizes the property viewing logistics by providing a luxury-tier interface for tenants to discover properties, schedule appointments, and securely upload legal identity documents via a cloud-native architecture.

### 🛠️ Technical Stack & Architecture
- **Infrastructure**: Vercel (Edge Runtime) + Supabase (BaaS).
- **Frontend**: Next.js 15, Tailwind CSS 4, Framer Motion.
- **Database**: PostgreSQL with Row Level Security (RLS).
- **Security**: Private Storage Buckets with Signed URL delivery for sensitive ID documents (Mission 1 Compliance).
